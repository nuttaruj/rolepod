# zoom-out

Compatibility shim. Zoom-out / re-orient discipline now lives in **`manage-context`**.

→ Open `core/skills/manage-context/SKILL.md` and follow that instead.

This shim preserves the legacy trigger phrase during the migration release.

## If `manage-context` is not available

Minimum viable fallback:

1. Step back from the immediate edit
2. Re-read the original user request literally
3. Ask: what is the user actually trying to accomplish?
4. Compare the current attempt path against the original goal
5. If you are solving a sub-problem you invented, stop — that is drift
6. List what you have tried and what failed; surface to the user if needed
7. Resume only after the actual goal is restated, not a derived one
