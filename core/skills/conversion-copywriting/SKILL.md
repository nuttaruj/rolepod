# conversion-copywriting

Compatibility shim. Conversion copy now lives in **`implement-plan`**; the `growth-marketer` agent adds depth when installed.

→ Open `core/skills/implement-plan/SKILL.md` and follow that instead. Brief `growth-marketer` if available.

This shim preserves the legacy trigger phrase during the migration release.

## If `implement-plan` is not available

Minimum viable fallback:

1. Headline = single benefit, in the reader's own language
2. One CTA per surface — do not split attention with multiple asks
3. Proof: testimonial, number, screenshot, named customer
4. Address one specific objection above the fold
5. CTA is an action verb (`Start free trial`, not `Learn more`)
6. Match the channel — email subject line, ad headline, landing hero each have different cadence
7. Measure clicks / signups / sales, not reads
